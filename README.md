## 新的部署流程
随着 github pages 服务的不断更新，现在终于不用折腾部署了。

删掉了之前用来测试各种部署方式的分支，其实模版带的脚本文件啥的也可以删了。

现在改为只保留 master 分支，并使用该分支下的*/doc*目录直接发布。

也就是说每次本地写完、调整完排版和错别字之后，构建一下，直接推送就好。
